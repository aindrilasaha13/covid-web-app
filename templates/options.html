Visit this space for modelling and predicting Covid-19 number of cases, using Machine Learning techniques such as Linear regression, Polynomial regression, Support Vector regression, Random Forest regression and Multi Layer Perceptron. Here you can enter the number of days for which you want to model and predict the number of Covid-19 cases. After that you will be able to view the prediction accuracy of each machine learning algorithm and also view the graph of actual data vs predicted data plotted for the chosen number of days.


Visit this space for viewing existing Covid-19 data for different states in India. You have to select the state for which you want to view the data, you have to also select the number of days for which you want to view the data. Then the confirmed number of Covid-19 cases, number of active cases, number of recovered cases and number of mortality cases for the chosen state for the chosen number of days will be displayed to you. This data will be displayed to you in a tabular form. You can also chose to view graphical representation of the same.


Visit this space for viewing existing Covid-19 data of all states in India. You have to select the date for which you want to view the data.Then the confirmed number of Covid-19 cases, number of active cases, number of recovered cases and number of mortality cases for the chosen number of days for all states in India will be displayed to you. This data will be displayed to you in a tabular form. You can also chose to view graphical representation of the same.
